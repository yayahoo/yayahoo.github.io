dataclass

参数 类型 说明
default 任意值 定义不可变类型的默认值（如数字、字符串）。
default_factory 可调用对象 定义可变类型的默认值。必须是一个函数（如 list, dict, set）。
init bool 默认为 True。如果设为 False，该字段不会出现在 init 的参数列表中。
repr bool 默认为 True。如果设为 False，在 print 该对象时不会显示此字段。
compare bool 默认为 True。如果设为 False，在进行对象比较（如 ==）时会忽略此字段。
metadata dict 存储第三方库或框架使用的元数据（对 Python 本身无特殊含义）。
  • default：存的是值（Value）。所有的实例都指向这同一个值。
  • default_factory：存的是函数（Function）。每个实例创建时，都会运行一遍这个函数来生成新值。
  • 当你给一个类加上 @dataclass 装饰器时，Python 会自动为你生成一个 __init__ 函数。在这个自动生成的函数末尾，它硬编码了一行调用：self.__post_init__()。

random

  1. random.sample(population, k)
  • population：可迭代对象，采样总体
  • k：采样数量
  • 返回：一个列表。不放回采样
  1. random.choices(population, k)
  • population：可迭代对象，采样总体
  • k：采样数量
  • 返回：一个列表。放回采样

list

  1. list.extend(iterable)
  • iterable：可迭代对象，依次加入到list中
  • 返回：None

内置函数

  1. getattr(obj, name, default=None)
  • obj：对象
  • name：属性名
  • default：默认值
  • 返回：属性值，如果属性不存在则返回默认值
